# Task 3

## Условие

Напишите программу с тремя переменными: одну с именем `b` типа `byte`, одну с
именем `smallI` типа `int32` и одну с именем `bigI` типа `uint64`.
Присвойте каждой переменной максимальное допустимое значение для ее типа,
затем добавьте `1` ко всем трем. Выведите их значения.

## Запуск

```bash
make run
```
