# Task 2

## Условие

Напишите программу, в которой объявлена константа `value`, которая может быть
присвоена как целочисленной переменной, так и переменной с плавающей запятой.
Присвойте ее целому числу `i` и переменной с плавающей запятой `f`.
Выведите значения `i` и `f`.

## Запуск

```bash
make run
```
